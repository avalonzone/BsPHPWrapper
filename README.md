# BsPHPWrapper
BsPHPWrapper is Framework/Wrapper aimed at easing the creation of Boostrap v3.3.7 interfaces by adding a layer of abstraction.

Include this autoloader file in your project : BsPHPWrapperAutoload.php

Things you need :
 - PHP 7
 - JQuery 3.2.0
 - Bootstrap 3.3.7
 - Datatable v1.10.18

For the moment :

 - No example
